My Journey So Far...

I started learning web development in 2022 after a conversation with a colleague, whose son recently started their career in programming.

My curiousity was piqued and I was looking for a new career path away from the medical and care sector I was in, so I started learning how to build websites and program their functionality using Javascript.

I have a degree in Illustration with Animation, majoring in drawing and narrative illustration for my final year project, but I also developed skills using editing and design software such as adobe Illustrator, Photoshop and inDesign which have proven quite useful in web design.

This site was built to showcase three projects that I created using HTML, CSS and Javascript and also to demonstrate the skills I have acquired so far on my learning journey. I hope you enjoy navigating this site and seeing some of the other projects I've worked on.
